1. Πως μπορούμε να κάνουμε ένα Request σε ένα εξωτερικό API χωρίς να γνωρίζει ολόκληρο το url μας; 
Απάντηση: Θα χρησιμοποιήσουμε εναν proxy server, έναν ενδιάμεσο server και έτσι o server του API 
θα έβλεπε το url του proxy server και οχι του client, δηλαδή το δικό μας.
		  
2. Αν κάνω ενα GET request και ζητήσω ενα jpeg image, το response τι content-type πρεπει να εχει; 
Απάντηση: Το content-type στο response θα είναι το image/jpeg

3. Πως μπορώ να κάνω ένα GET request και το περιεχόμενο του να κρατηθεί στην cache για 1 μέρα; 
Απάντηση: Θα χρησιμοποιήσουμε το HTTP Header Cache-Control με max-age=86400 (σε seconds)

4. Μπορώ να έχω μια εφαρμογή που κάνει μόνο GET requests; 
Απάντηση: Ναι, μπορεί μια εφαρμογή να κάνει μόνο GET requests και να λαμβάνει πράγματα που ήδη υπάρχουν
		  στον server, αλλά να μην δημιουργεί κάτι ή να προκαλεί αλλαγές στα υπόλοιπα.
		  
5. Μπορώ να έχω μια εφαρμογή που κάνει μόνο POST requests; 
Απάντηση: Ναι, μπορεί μια εφαρμογή να κάνει μόνο POST requests και να δημιουργεί κάτι καινούργιο 
		  ή να υποβάλλει πχ τα credentials σε μια login form.
		  
6. Τι status code περιμένω από ένα API αν δεν έχω στείλει ένα υποχρεωτικό πεδίο; 
Απάντηση: Αν δεν έχω στείλει ένα υποχρεωτικό πεδίο θα πρέπει να περιμένω το error 400: Bad Request
		  επειδή ο server θα περιμένει να λάβει δεδομένα εκεί και δεν υπάρχουν στο request.
		  
7. Τι status code περιμένω από ένα API αν δεν υπάρχει επικοινωνία με τη βάση; 
Απάντηση: Αν δεν υπάρχει επικοινωνία με τη βάση τότε θα περιμένω το error 500: Internal Server Error

8. Μπορώ να κάνω Login με GET request; 
Απάντηση: Όχι δεν μπορώ να κάνω Login με GET request γιατί θέλω να στείλω τα credentials στον server, 
		  ενώ με την GET λαμβάνω δεδομένα. Για να κάνω login και να στείλω τα credentials 
		  θα χρησιμοποιήσω το POST.

9. Ποια είναι η διαφορά ανάμεσα στο PUT και στο POST; 
Απάντηση: Θα χρησιμοποιήσουμε το PUT για να κάνουμε update σε κάτι που ήδη υπάρχει στον server
          ενώ το POST για να δημιουργήσουμε κάτι καινούργιο πχ ένα καινούργιο record ή να ανεβάσουμε ένα αρχείο.
		  Επίσης την χρησιμοποιούμε για στείλουμε form data πχ όταν κάνουμε login.

10. Ποια μέθοδο θα πρέπει να χρησιμοποιήσω για να «ανεβάσω» ένα αρχείο;
Απάντηση: Για να ανεβάσω ένα αρχείο θα χρησιμοποιήσω την POST.
